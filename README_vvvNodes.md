# vvvNodes for ComfyUI

Коллекция вспомогательных узлов для ComfyUI, включающая продвинутую работу с JSON, кастомные слайдеры и исполнитель Python-кода.

---

## 1. Работа с JSON (JSON pipe)

Узлы для извлечения даных и манипуляции JSON-объектами по заданному пути (`key_path`). Путь указывается через точку (например, `user.profile.name`).

### JSON pipe (vvvNodes)
**Строгий режим** для быстрого извлечения обязательных данных.
- **Логика**: Если указанный `key_path` отсутствует в JSON, выполнение workflow **останавливается с ошибкой**.
- **Входы**: 
    - `pipe (any)`: Входящий JSON (словарь или строка).
- **Выходы**:
    - `pipe (any)`: Тот же JSON-объект для дальнейшей передачи.
    - `value (any)`: Извлеченное значение.

### JSON pipe (try get) (vvvNodes)
**Мягкий режим** для опциональных данных.
- **Логика**: Если путь не найден, возвращает `None` и **продолжает** выполнение.
- **Выходы**:
    - `pipe (any)`: Весь JSON-объект.
    - `value (any)`: Значение или `None`.
    - `found`: BOOLEAN (True, если путь успешно найден).

### Universal JSON pipe (vvvNodes)
Максимально гибкий узел для чтения, записи и отладки JSON.

- **Чтение и Запись**:
    - Если задействован вход `value (any)` или заполнен виджет `value`, узел **записывает** это значение в JSON по пути `key_path` (создавая структуру папок, если нужно).
- **Key Path**: 
    - Если оставить поле **пустым**, узел вернет и покажет в Preview **весь входной JSON**.
- **Default on read**: Значение по умолчанию, если путь не найден.
- **Логика остановки (Strict Mode)**:
    - Узел выдаст ошибку, только если путь не найден **И** оба поля подтверждения (`default_on_read` и виджет `value`) абсолютно пустые. 
    - Чтобы явно разрешить пустую строку как fallback, впишите в виджет `""` (две кавычки).
- **Preview (Индикация)**:
    Узел выводит результат в текстовый блок с префиксом источника:
    - `found: ` — значение прочитано из JSON.
    - `value: ` — значение было записано/обновлено в этом узле.
    - `default: ` — использовано значение по умолчанию.

## 2. Слайдеры (Custom Slider)

**🎚️ Custom Slider (vvvNodes)**
Заменяет стандартные слайдеры ComfyUI, добавляя функционал:
- **Precision**: Математическое округление результата (0 для целых чисел).
- **Slider Color**: Выбор цвета для визуального разделения параметров.
- **Tooltip**: Текст подсказки, отображаемый в интерфейсе.
- **Direct Entry**: Возможность ввода точного числа через двойной клик.

---

## 3. Скриптинг (Python Executor)

**🐍 Python Code Executor (vvvNodes)**
Нода для реализации сложной логики на лету без создания новых файлов.
- **Окружение**: Внутри доступны библиотеки `torch`, `math`, `numpy` (как `np`).
- **Интерфейс**:
    - Входы `in1`, `in2`, `in3` доступны в коде как переменные.
    - Результаты нужно присвоить переменным `out1` и `out2`.
- **Пример**:
  ```python
  out1 = np.mean(in1) if in1 is not None else 0
  out2 = in2 * math.pi
  ```

## Установка
Просто скопируйте папку `ComfyUI-vvvNodes` в директорию `custom_nodes` вашего ComfyUI.
